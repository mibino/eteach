# ETeach 教学工具
## 这是什么？
ETeach是一个来源的教学工具软件，使用Python编写\
## 它支持什么操作系统？
目前支持Windows,Linux,MacOS
## 我要怎么安装并使用它？
1.下载Python(3.8以上)
2.运行install.py文件安装所需要的Python第三方库
3.运行app.py(可自行创建快捷方式)
## 它目前有什么功能？
简易的白板功能\
太空模拟功能
## 未来有什么功能？
物理模拟器\
高级计算器
## 我要怎么向这个项目贡献自己的代码？
很简单，你可以修改完代码后提交一个PR，或在Issues提出一个建议
## 我要怎么支持一个项目？
可以给个Star，也可以宣传这个项目
